Installation
------------

```console
$ curl -kL https://raw.github.com/yonchu/dotfiles/master/etc/setup.sh | bash
```
or

```console
$ cd
$ git clone --recursive https://github.com/yonchu/dotfiles.git
$ cd dotfiles && ./etc/setup.sh
```
